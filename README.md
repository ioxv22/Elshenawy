# موقع الأستاذ إسلام الشناوي - مدرس فيزياء متخصص

## نظرة عامة
موقع تعليمي عصري ومتطور للأستاذ إسلام الشناوي، مدرس الفيزياء للصفوف المتقدمة في الإمارات. يقدم الموقع تجربة تعليمية شاملة تشمل الكورسات، التجارب العلمية التفاعلية، وشهادات الشكر.

## الميزات الرئيسية

### 🎨 التصميم العصري
- ألوان متناسقة ومريحة للعين
- واجهة جذابة وسهلة التنقل
- خطوط واضحة وسهلة القراءة (Cairo & Poppins)
- دعم كامل للتصفح على الهواتف والأجهزة اللوحية
- وضع مظلم/فاتح قابل للتبديل

### 📚 المحتوى التعليمي
- **الكورسات**: مجموعة شاملة من الفيديوهات التعليمية
- **التجارب العلمية**: تجارب تفاعلية للصف العاشر المتقدم
- **شهادات الشكر**: معرض لشهادات التقدير والإنجازات
- **نظام الاشتراكات**: خطط مختلفة للطلاب

### 🔬 التجارب العلمية التفاعلية
- **Pendulum Simulation Lab**: محاكاة البندول
- **Mass and Spring Simulation**: محاكاة الكتلة والنابض
- **Wave on a String Simulation**: محاكاة الموجات على الخيط
- **Waves Resources**: مصادر إضافية للموجات

### ⚡ الأداء والتفاعل
- تحميل سريع ومحسن
- أنيميشن متقدمة وسلسة
- تفاعل محسن مع المستخدم
- دعم اختصارات لوحة المفاتيح
- شريط تقدم القراءة
- تحميل كسول للصور

### 🎯 تجربة المستخدم
- تنقل سلس بين الأقسام
- بحث متقدم في الكورسات
- نظام تقييم وآراء الطلبة
- نماذج اتصال محسنة
- دعم إمكانية الوصول

## التقنيات المستخدمة

### Frontend
- **HTML5**: هيكل الموقع
- **CSS3**: التصميم والتنسيق
  - CSS Grid & Flexbox
  - CSS Variables
  - Animations & Transitions
  - Media Queries للاستجابة
- **JavaScript (ES6+)**: التفاعل والوظائف
  - Intersection Observer API
  - Local Storage
  - Event Handling
  - DOM Manipulation

### المكتبات والأدوات
- **Font Awesome**: الأيقونات
- **Google Fonts**: الخطوط (Cairo, Poppins)
- **AOS**: أنيميشن التمرير
- **PWA**: دعم تطبيق الويب التقدمي
